После покупки получаете архив с ботом. В самом архиве будет:
1. Сам файл бота.
2. Файл install.bat - после запуска автоматически установит все отсутствующие модули.
3. В самом файле с ботом нужно установить:
admin_id = тут свой айди. его можно узнать - https://t.me/getmyid_bot
token_api = "ТОКЕН" . Его можно узнать тут - https://t.me/BotFather , create a new bot, api token.
4. Запуск бота.
